Glitch-学js的新玩具

发表于 2017-12-19 | 分类于 web

Glitch 是一个简单的构建 Javascript web 程序的方式（javascript 后端、javascript 前端）。-- Julia Evans

Glitch是一个帮助程序员建立理想应用程序的友好社区，对于初学者来说，这个社区简直就像及时雨。

他们可以对Glitch中所有的酷炫项目进行调整，做出符合自己想象的网站。

关于Glitch有趣的在于

☉ 你在他们的网站输入 Javascript 代码
☉ 只要输入了任何代码，它会自动用你的新代码重载你的网站。它甚至会自动保存。

Glitch 是 Fog Creek Software 推出的一个新式协作编程网站，该公司还曾打造过 Trello 和 FogBugz 。

Glitch 是在 Anil Dash 带领开发的，他们的目标很简单 ——“提供一个友好的社区，让你可以在这里构建一款梦想中的应用”。

“Gomix”是 Glitch 在公测时所使用过的名字，其主要功能是将现有项目代码混搭起来，让用户能够轻松用上早已在其它应用程序、网站、机器人中实现的部分功能，然后再进行优化。

在创建应用程序时，Glitch允许程序员进行协作。

Glitch不需要复杂的版本控制，内置的编辑器就足以满足程序员和其他人编辑代码的需求，在发生错误时还可以撤销。还有一个重要且实用的功能，那就是Glitch可以将代码直接导入和导出到GitHub。

公众开放

Glitch团队明确表示，Glitch不是一个真正的开发人员环境的“dumbed down”版本。实际上，它与所有程序员在实际环境中用于他们的应用程序的任何其他云基础设施+ Node.js引擎.

Enjoy it and have fun.
